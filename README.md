# Project Name

A Python project template.

## Setup

### 1. Create a virtual environment
```bash
python -m venv .venv
```

### 2. Activate the virtual environment
**Windows:**
```bash
.venv\Scripts\activate
```

**macOS/Linux:**
```bash
source .venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

## Running the project

```bash
python app.py
```

```
.
├── src/              # Source code
```
