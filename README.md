## ✨ Features
- Upload any CSV file (supports 400,000+ rows)
- Interactive data table with stats
- Dynamic charts — Histogram, Bar Chart, Line Chart
- AI-powered insights generated by LLaMA 3 (Groq API)
- Clean sidebar UI with how-to guide

## 🛠️ Built With
- Python
- Streamlit
- Pandas
- Matplotlib & Seaborn
- LLaMA 3 via Groq API

## 🚀 How to Run Locally
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
4. Get a free API key from groq.com and paste it in the app
